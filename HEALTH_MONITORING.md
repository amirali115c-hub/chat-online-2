# Website Health & Analytics

This module monitors your website for hidden issues and provides analytics.

## Features

### 1. Performance Monitoring
- Page load times
- Slow pages detection
- Server response times

### 2. Health Checks
- Database connectivity
- Socket.IO connections
- Memory usage
- Active sessions

### 3. Error Tracking
- JavaScript errors
- 404 pages
- Failed requests
- Login failures

### 4. SEO Analysis
- Meta tags missing
- Broken internal links
- Image alt text missing

### 5. Security Monitoring
- Failed login attempts
- Suspicious activity
- Rate limit violations

### 6. User Analytics
- Page views
- Unique visitors
- Popular pages
- Traffic sources

## Usage

Access the health dashboard at `/admin/health`

Or view specific reports:
- `/api/health` - JSON health status
- `/api/health/performance` - Performance metrics
- `/api/health/errors` - Recent errors
- `/api/analytics` - User analytics

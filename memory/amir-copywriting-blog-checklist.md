## WRITING PHASE - STRUCTURE

### 3. Meta Elements
- [ ] Meta title (55-60 chars, primary keyword front-loaded, year)
- [ ] Meta description (150-160 chars, keyword, credibility, soft CTA)
- [ ] URL slug (3-6 words, primary-keyword-context-year)

### 4. Introduction (Opening Hook Formula - 200-300 words)
- [ ] Casual opener ("Look, I'm gonna be straight with you")
- [ ] One-sentence entity introduction with hook
- [ ] Personal experience ("I've been working with [entity] for [time]")
- [ ] Contrarian statement ("And honestly? [Surprising claim]")
- [ ] Rhetorical question + short answer + explanation
- [ ] Entity context with semantic keywords
- [ ] What you're gonna learn list (5 items)

### 5. Content Body (2000-3000 words total)
- [ ] 7-10 H2 sections with entity-focused titles
- [ ] Each H2: 350-500 words
- [ ] H2 Formula: [How/Why] + [Entity/Attribute] + [Specific Benefit] + [Personal Credibility]

### 6. H2 Structure (Per Section)
- [ ] Casual opener ("Okay, so", "Look", "Alright")
- [ ] Short sentence + contraction
- [ ] Personal statement + experience
- [ ] Entity definition paragraph
- [ ] Data/research with 2025-2026 citations
- [ ] Real-world application with first-hand experience
- [ ] Actionable framework

### 7. FAQ Section
- [ ] 8-12 questions from Google's People Also Ask
- [ ] Mix beginner + advanced entity questions
- [ ] 60-120 words per answer
- [ ] Personal voice + contractions

### 8. Conclusion
- [ ] Key takeaways (5 bullet points)
- [ ] Reinforcement paragraph
- [ ] Dual CTAs (soft + hard)
- [ ] Forward hook

### 9. About the Author
- [ ] 150-200 words
- [ ] First-person
- [ ] Credentials included
- [ ] Personality showing

## WRITING PHASE - VOICE & STYLE

### 10. Banned Patterns Check
- [ ] No formal connectors ("Despite", "Nevertheless", "Furthermore", "Moreover")
- [ ] No academic language
- [ ] No problem listing (no multiple sentences about same problem)
- [ ] No Question + Bridge + Statement formula
- [ ] No parallel sentence pairs

### 11. Mandatory Voice Elements
- [ ] Personal experience ("I have seen", "I have talked to", "In my experience")
- [ ] Honest frustration/directness
- [ ] Simple contrasts ("Their budgets grow. Their results do not.")
- [ ] Fragments for impact ("And most of it? Wasted.")
- [ ] Summary statements at end of paragraphs

### 12. Amir's Soul Check
- [ ] Specific client references (real names, real results)
- [ ] Dubai/UAE context feels lived, not researched
- [ ] Confident without arrogance
- [ ] Has opinions about the topic
- [ ] References own experience naturally

### 13. Contractions (95%+ MANDATORY)
- [ ] don't, can't, won't, it's, I've, I'm, you're
- [ ] we've, they're, hasn't, doesn't, wouldn't
- [ ] couldn't, shouldn't, hasn't, doesn't

### 14. Casual Sentence Starters (50%)
- [ ] Look, So, And, But, Well, Now
- [ ] Okay, Yeah, See, Plus, Actually
- [ ] Honestly, Basically, Really, Just

### 15. Human Writing Elements
- [ ] 8-12 sentence fragments for impact
- [ ] 5-8 rhetorical questions
- [ ] 5-8 parenthetical asides
- [ ] Natural word repetition (humans repeat)
- [ ] Intentional grammar breaks

### 16. Sentence Length Variety
- [ ] 4 SHORT (3-7 words): Impact, emphasis
- [ ] 4 MEDIUM (10-18 words): Context, clarity
- [ ] 2 LONG (25-45 words): Complex entity relationships
- (Repeat pattern every 10 sentences)

### 17. Paragraph Length Variety
- [ ] 20% one-sentence paragraphs
- [ ] 40% two-sentence paragraphs
- [ ] 30% three-to-four-sentence paragraphs
- [ ] 10% five-plus-sentence paragraphs

## OPENING PARAGRAPH FORMULA (ALWAYS USE)

```
[Solution A]
Every single year, [context]. And [fragment]? [shock]. 
I have seen [personal experience]. No [outcome 1]. No [outcome 2]. Just [outcome 3].

[Solution D]
In [location], [industry] is [description]. [expansion]. 
Yet most [people I have talked to] feel [honest observation]. 
Their [positive]. Their [negative]. This [summary statement].
```

## QUICK REFERENCE - EM DASH BAN

**NEVER USE: —**

Examples of what NOT to write:
- "Most advice is garbage — and I've seen it fail"
- "Something is broken — nobody tells you this"
- "This disconnect — the real problem"

Instead write:
- "Most advice is garbage. And honestly? I've seen it fail."
- "Something is broken. Nobody tells you this."
- "This disconnect. This is the real problem."

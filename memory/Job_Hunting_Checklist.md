# Job Hunting & Application Checklist — Tool-Enhanced Strategy

## Overview
Leverage NightOwl, N8N, OpenAI Operator, Lindy AI, and Gumloop for automated, efficient job hunting.

---

## PHASE 1: JOB DISCOVERY & RESEARCH

### 1.1 Target Company Identification
- [ ] Define target industries and roles
- [ ] Identify 50-100 target companies
- [ ] Research company culture and values
- [ ] Map company hiring cycles (Q1, Q2, etc.)
- [ ] Identify decision-makers and recruiters on LinkedIn
- [ ] Set up company news alerts

### 1.2 Job Board Monitoring
- [ ] LinkedIn Jobs alerts set up (keywords, locations, experience levels)
- [ ] Indeed daily alerts configured
- [ ] Glassdoor job notifications active
- [ ] BuiltInTech and remote-specific boards covered
- [ ] Company career pages bookmarked
- [ ] N8N workflow: Aggregate jobs from all sources daily
- [ ] NightOwl task: Weekly SERP analysis for "hiring now" signals

### 1.3 Opportunity Research
- [ ] Research each job posting thoroughly
- [ ] Identify must-have vs nice-to-have requirements
- [ ] Map role to career trajectory
- [ ] Research interviewer profiles on LinkedIn
- [ ] Analyze company reviews (Glassdoor, Blind, Fishbowl)
- [ ] Identify red flags (high turnover, bad culture)
- [ ] Research salary ranges (Levels.fyi, Glassdoor, Blind)

## PHASE 3: APPLICATION EXECUTION

### 3.1 Application Tracking
- [ ] Spreadsheet tracker created (company, role, URL, status, date, contacts)
- [ ] Status categories: Applied, In Progress, Interviewing, Offer, Rejected, Closed
- [ ] Follow-up dates marked (7-14 days post-application)
- [ ] Response tracking (response rate, interview rate)
- [ ] N8N workflow: Auto-log applications from emails
- [ ] Lindy AI: CRM integration for all applications

### 3.2 Application Submissions
- [ ] Easy Apply opportunities (LinkedIn, Indeed) - 10/day target
- [ ] Company career pages - 5/day target
- [ ] Referral requests - 10/week
- [ ] Recruiter outreach - 5/day
- [ ] OpenAI Operator: Automated form filling
- [ ] Gumloop: Bulk application workflow automation
- [ ] Cover letters attached (where applicable)
- [ ] Portfolio links included
- [ ] LinkedIn profile URL included everywhere

### 3.3 Referral Strategy
- [ ] Warm referral targets identified (2nd degree connections)
- [ ] Personalized outreach message template ready
- [ ] Value proposition for each connection defined
- [ ] Follow-up sequence (Day 1, Day 3, Day 7)
- [ ] LinkedIn InMail strategy for non-connections
- [ ] Alumni network leverage activated
- [ ] Lindy AI: Automate referral request outreach

## PHASE 4: INTERVIEW PREPARATION

### 4.1 Company Research
- [ ] Company website thoroughly reviewed
- [ ] Recent news and press releases read
- [ ] LinkedIn company page analyzed
- [ ] Competitors identified and studied
- [ ] Products/services thoroughly understood
- [ ] Leadership team researched
- [ ] Funding status and company health checked
- [ ] NightOwl task: Deep dive company SERP analysis

### 4.2 Role Preparation
- [ ] Job description deconstructed
- [ ] Key competencies identified
- [ ] 5 STAR stories prepared for common questions
- [ ] Technical requirements reviewed
- [ ] Common interview questions answered
- [ ] Role-specific knowledge gaps identified
- [ ] NightOwl: Audit competitor interview prep materials

### 4.3 Technical Preparation
- [ ] LeetCode/HackerRank practice schedule (daily 1-2 hours)
- [ ] Key algorithms and data structures reviewed
- [ ] System design concepts studied
- [ ] Role-specific technical prep (languages, frameworks)
- [ ] Past projects reviewed and documented
- [ ] Code samples prepared (clean, documented)
- [ ] Take-home assignments strategy ready

### 4.4 Mock Interviews
- [ ] At least 2 mock interviews scheduled
- [ ] Peer interview partners identified
- [ ] Interview recording setup
- [ ] Feedback loop established
- [ ] Weaknesses identified and addressed
- [ ] Exponent/Pramp sessions completed

### 4.5 Interview Logistics
- [ ] Timezone conversions confirmed
- [ ] Interview links tested (Zoom, Teams, etc.)
- [ ] Environment prepared (quiet, good lighting, backup internet)
- [ ] Professional attire selected
- [ ] Questions for interviewer prepared (5-7)
- [ ] Thank you email template ready

---

## PHASE 5: INTERVIEW EXECUTION

### 5.1 Pre-Interview
- [ ] Night before: Review company and role
- [ ] Prepare outfit and tech setup
- [ ] Have water and notes nearby
- [ ] Practice STAR stories aloud
- [ ] Final LinkedIn review of interviewers
- [ ] Research panel members if known

### 5.2 During Interview
- [ ] Arrive 5 minutes early (virtual)
- [ ] Strong handshake or confident greeting
- [ ] Body language open and engaged
- [ ] STAR method for behavioral questions
- [ ] Ask prepared questions
- [ ] Take notes on key points
- [ ] Request next steps and timeline
- [ ] Express enthusiasm for role

### 5.3 Post-Interview
- [ ] Send thank you within 24 hours (personalized)
- [ ] Reference specific conversation points
- [ ] Reiterate interest and fit
- [ ] Update application tracker
- [ ] Note any red flags or concerns
- [ ] Follow up if timeline passes (Day 7-14)
- [ ] Lindy AI: Automate thank you email sequences

---

## PHASE 6: OFFER EVALUATION & NEGOTIATION

### 6.1 Offer Analysis
- [ ] Base salary compared to market (Levels.fyi, H1B data)
- [ ] Bonus structure evaluated (signing, annual, performance)
- [ ] Equity/RSUs reviewed (vesting schedule, strike price)
- [ ] Benefits compared (health, 401k, PTO, etc.)
- [ ] Relocation package assessed
- [ ] Remote work policy clarified
- [ ] Total compensation calculated (TC)

### 6.2 Negotiation Preparation
- [ ] Target and walk-away numbers defined
- [ ] Comparable offers gathered
- [ ] BATNA established (Best Alternative to Negotiated Agreement)
- [ ] Negotiation script prepared
- [ ] Timing of negotiation decided
- [ ] OpenAI Operator: Research market rates and offers
- [ ] NightOwl: SERP analysis for negotiation tactics

### 6.3 Negotiation Execution
- [ ] Express enthusiasm before negotiating
- [ ] Present data-backed requests
- [ ] Consider full package (not just base)
- [ ] Be willing to walk away
- [ ] Get everything in writing
- [ ] Deadline for decision established

## PHASE 9: METRICS & OPTIMIZATION

### 9.1 Key Metrics to Track
- [ ] Applications sent per week
- [ ] Response rate (target: 10-15%)
- [ ] Interview rate (target: 20-30% of responses)
- [ ] Offer rate (target: 10-15% of interviews)
- [ ] Time-to-offer average
- [ ] Offer acceptance vs rejection ratio
- [ ] Company type breakdown (startup vs enterprise)
- [ ] Role level breakdown

### 9.2 Funnel Analysis
- [ ] Applications → Recruiter Screen
- [ ] Recruiter Screen → Hiring Manager Interview
- [ ] Hiring Manager → Technical Interview
- [ ] Technical → Final Round
- [ ] Final Round → Offer
- [ ] Identify bottlenecks in funnel

### 9.3 Continuous Improvement
- [ ] After each rejection: Request feedback
- [ ] After each interview: Note improvement areas
- [ ] Monthly resume iteration
- [ ] Weekly strategy adjustment
- [ ] A/B test cover letter approaches
- [ ] Track which channels yield best results

## COMMON PITFALLS TO AVOID

- [ ] Applying to every job (focus on quality)
- [ ] Neglecting networking (70% of jobs are hidden)
- [ ] Generic resume and cover letter
- [ ] Not following up after applications
- [ ] Ignoring red flags in interviews
- [ ] Accepting first offer without negotiation
- [ ] Neglecting technical preparation
- [ ] Burning out (take breaks!)
- [ ] Limiting to one channel (diversify!)
- [ ] Giving up too early (persistence wins)

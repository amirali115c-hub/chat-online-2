**Location:** D:\Chat Online

### Project Overview
Flask-based anonymous chat application with real-time messaging using Socket.IO.

### Tech Stack
- **Backend:** Flask, Flask-SocketIO, SQLite
- **Frontend:** HTML/CSS/JavaScript templates
- **Security:** CSRF protection, rate limiting, bot detection, honeypot verification

### Key Features
1. **Anonymous Chat** - Random partner matching based on gender preferences
2. **User Registration/Login** - Username, gender, age, country, state
3. **Friend System** - Add friends, friend requests, friends list
4. **Chat Rooms** - Multiple chat modes (1on1, random, dating channels)
5. **Real-time Messaging** - WebSocket-based with typing indicators
6. **Admin Dashboard** - User management, message moderation, reports
7. **Rate Limiting** - Anti-bot protection (max 20 msgs/min, 30 req/min)
8. **Message Limits** - 100 messages per chat session
